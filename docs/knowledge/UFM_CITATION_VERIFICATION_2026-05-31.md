## Substantive findings that sharpen the engine
- **Exhibit index forks by record type.** Official (trial) 3.23(c): description + page *offered and received into evidence*. Freelance (deposition) 3.24(a)(6): description + page *referenced or marked* only. → offered/admitted prints for trials, not depositions.
- **Rough draft (4.4):** must NOT contain format box, title page, appearance page, certification, or index. A rough-draft package = body + disclaimer (4.5) only.
- **Index spacing (3.23(e)):** columnar; single-spaced within a witness's exams, double-spaced between topic changes; the three indexes flow onto the same page if space allows (matches Shaw's consolidated front matter).
- **Tab divergence confirmed:** UFM mandates exactly 3 tabs (5th/10th/15th). checkpoint_A found the live profile exposes 5 — real divergence to reconcile.
